# 개인정보처리방침 (Privacy Policy)

**Gemini 목차 생성기 (Gemini ToC Maker)** 확장 프로그램은 사용자의 개인정보를
소중하게 생각하며, 다음과 같은 방식으로 데이터를 처리합니다.

## 1. 수집하는 데이터 및 처리 방식 (Data Collection and Processing)

본 확장 프로그램은 개발자의 서버로 사용자의 데이터를 **수집하거나 전송하지
않습니다.**

단, 기능 동작을 위해 다음과 같은 데이터를 처리합니다:

- **채팅 내용 (Chat Content)**: 목차 생성을 위해 사용자가 보고 있는 Gemini
  채팅의 텍스트가 사용자의 브라우저에서 **Google Gemini API**로 직접 전송됩니다.
  이 데이터는 요약 목적으로만 사용되며, 개발자가 이를 별도로 저장하거나 조회하지
  않습니다.
- **Gemini API Key**: 사용자가 입력한 API Key는 사용자의 브라우저 내 로컬
  스토리지(`chrome.storage.local`)에만 안전하게 저장됩니다. 이 키는 요약 요청을
  보내는 용도로만 사용되며 외부로 유출되지 않습니다.

## 2. 제3자 서비스 (Third-Party Services)

이 확장 프로그램은 요약 기능을 제공하기 위해 다음 외부 서비스를 이용합니다:

- **Google Gemini API**: 채팅 텍스트를 분석하고 요약하기 위해 Google의 생성형 AI
  서비스를 사용합니다. Google의 데이터 처리 방식은
  [Google Privacy Policy](https://policies.google.com/privacy)를 따릅니다.

## 3. 데이터 보관 (Data Retention)

- **API Key**: 사용자가 확장 프로그램을 삭제하거나 브라우저 데이터를 삭제할
  때까지 브라우저 내에 저장됩니다. 사용자는 언제든지 설정 메뉴에서 키를
  변경하거나 삭제할 수 있습니다.
- **목차 데이터**: 생성된 목차는 일시적으로 메모리에만 존재하며, 페이지를
  새로고침하거나 닫으면 사라집니다. 별도의 데이터베이스에 저장되지 않습니다.
