# Gemini 목차 생성기 (ToC Maker)

[English Version (영어 버전)](./README.md)

**Gemini 목차 생성기**는 Google Gemini 채팅 내용을 바탕으로 자동으로 목차(Table
of Contents)를 생성해주는 크롬 확장 프로그램입니다. AI를 활용하여 긴 대화 내용을
요약하고 구조화하여, 원하는 내용을 쉽게 찾고 빠르게 이동할 수 있도록 도와줍니다.

![스크린샷 1](icons/screenShot.png) ![스크린샷 2](icons/screenShot2.png)

## ✨ 주요 기능

- **목차 자동 생성**: 채팅 히스토리를 분석하여 자동으로 목차를 만들어줍니다.
- **간편한 탐색**: 목차의 항목을 클릭하면 대화의 해당 위치로 즉시 스크롤됩니다.
- **테마 지원**: Gemini의 라이트 모드와 다크 모드에 맞춰 디자인이 자동으로
  적용됩니다.
- **API 키 설정**: 사용자의 Gemini API Key를 설정하여 요약 기능을 사용할 수
  있습니다.

## 🚀 설치 방법

크롬 웹 스토어에서 직접 설치할 수 있습니다:

[**👉 크롬 웹 스토어에서 다운로드**](https://chrome.google.com/webstore/detail/placeholder)

_(확장 프로그램이 게시되면 링크가 업데이트될 예정입니다)_

## ⚙️ 사용 방법

1. **API 키 설정**:
   - 브라우저 툴바에서 확장 프로그램 아이콘을 클릭합니다.
   - 설정 팝업창에 Google Gemini API Key를 입력합니다.
   - "Save Settings" 버튼을 클릭하여 저장합니다.
2. **Gemini 실행**:
   - [gemini.google.com](https://gemini.google.com)에 접속합니다.
   - 채팅을 엽니다.
   - 화면 우측 패널의 '목차 생성' 버튼을 클릭하여 목차를 확인하세요.
